# 최준용 보안 인프라 포트폴리오

보안 인프라 구축, 침투 검증, 통합 관제 및 운영 자동화 경험을 소개하는 포트폴리오 사이트입니다.

## 주요 내용

- Shielders 기업형 네트워크·Linux 서버 구축
- NextBank 침투테스트 및 통합 보안 관제
- 프로젝트 발표자료 기반 구성도와 관제 결과
- CTF 대회 2등 및 워게임 3등 수상 기록
- 프로젝트별 Notion 상세 문서 연동

## 로컬 실행

```bash
pnpm install
pnpm dev
```

## 빌드

```bash
pnpm build
```
